# QR Tally

College attendance system using rotating HMAC-signed QR codes.

## Stack

- React + Vite + TypeScript + Tailwind CSS
- Supabase (Postgres, Auth, Realtime, Edge Functions)

## Roles

In order of authority:

- Super Admin
- College Admin
- Head Coordinator
- Module Coordinator
- Teacher
- Student
- Coordinator (legacy)
